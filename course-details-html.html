Structuring with HTML

Learn the fundamentals of structuring web documents with HTML in this course.

The "Foundations of HTML for Web Development" course at ABC Learning Center is crafted for beginners who wish to learn the fundamental building blocks of web development. This course covers all essential aspects of HTML (HyperText Markup Language), the standard language for creating web pages. Participants will gain a thorough understanding of HTML syntax and structure, enabling them to develop well-organized and accessible web pages from scratch.

Course Details

Duration: 8 weeks

Cost: RM400

Start Date: 1 August 2024

End Date: 31 August 2025

Course Outline

Week 1: Introduction to HTML

Week 2: Creating Content

Week 3: Working with Multimedia

Week 4: Building Forms

Week 5: Tables and Data Representation

Week 6: Semantic HTML

Week 7: Linking and Navigation

Week 8: Best Practices

Learning Outcomes

By the end of this course, participants will be able to:

  • Understand the basic structure and syntax of HTML.
  • Create and structure web pages using HTML elements.
  • Embed multimedia elements like images, audio, and video.
  • Implement forms to collect user input.
  • Use semantic HTML to enhance accessibility and SEO.
  • Link web pages together and create navigation structures.
