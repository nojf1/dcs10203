Programming in .NET

Learn the fundamentals of .NET framework in this course.

The "Fundamentals of .NET Programming" course at ABC Learning Center is tailored for beginners and intermediate learners who aspire to master the .NET framework. This course covers the core concepts and components of .NET, enabling participants to develop robust, scalable, and secure applications. The curriculum provides a deep dive into C# programming, the primary language for .NET development, along with hands-on experience in building web and desktop applications.

Course Details

Duration: 7 weeks

Cost: RM400

Start Date: 1 August 2024

End Date: 31 August 2025

Course Outline

Week 1: Introduction to .NET Framework

Week 2: C# Basics

Week 3: Object-Oriented Programming (OOP) in C#

Week 4: Developing Web Applications with ASP.NET

Week 5: Database Connectivity with ADO.NET

Week 6: Exception Handling and Debugging

Week 7: Developing Desktop Applications

Learning Outcomes

By the end of this course, participants will be able to:

  • Understand the architecture and components of the .NET framework.
  • Write and execute C# programs efficiently.
  • Implement object-oriented programming principles using C#.
  • Develop web applications using ASP.NET.
  • Create desktop applications with Windows Forms and WPF.
  • Utilize ADO.NET for database connectivity and operations.
  • Handle exceptions and perform debugging to ensure application reliability.
